Train Sheduler

This web application allows users to create and store fictitious train schedules based on theis input and on the current time, which is displayed on the top of the page and updates every 60 seconds. It accepts parameters such as Train Name, Destination, First Train Time and Frequency. The information about the train will be displayed in a table and is stored in a database so it persists even after a page refresh.
