FunHub - Find fun near you!

This Event Finder web application helps users find events based on keyword input, location and category. The results displayed will allow the user to buy tickets through the vendor's website and to see a list of the nearest breweries within the event’s location.

Train Sheduler

This web application allows users to create and store fictitious train schedules based on theis input and on the current time, which is displayed on the top of the page and updates every 60 seconds. It accepts parameters such as Train Name, Destination, First Train Time and Frequency. The information about the train will be displayed in a table and is stored in a database so it persists even after a page refresh.

Rise and Grind

In this fun web application the user is able to generate animated GIFs on the click of a button. The app gives 10 buttons representing the steps of a daily grind. IF the user clicks on any of the default buttons it will generate 10 animated GIFs and its respective rating. The Gifs can be paused and become static if the user clicks on the image, and will return to the original state if clicks once again. Another feature the app offers is the possibility to create new buttons so the user can customize the ideal daily grind.
